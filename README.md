# Matching-kafka

## 개요 
- Kafka를 기반으로 설계된 1:1 매칭 시스템입니다. <br>
  // 현재는 실제 서비스에서 사용되지 않으며, 참고용으로 보존하고 있습니다.

<br>

## 주요 기능
- 사용자 간 1:1 매칭 서비스
- 관심사 및 위치 기반 매칭
- Kafka 기반 큐 관리
